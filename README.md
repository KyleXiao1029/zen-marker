# ZenMarker

ZenMarker 是一个基于 React 和 Vite 构建的现代化 Chrome 书签管理扩展，采用 Apple 风格的精美设计。旨在提供比原生书签管理器更高效、更美观的使用体验。

## ✨ 主要功能

- **精美界面**: 采用现代化、极简的 Apple 风格 UI 设计，视觉体验舒适。
- **书签管理**:
  - 支持创建、编辑、删除书签和文件夹。
  - 侧边栏树形导航，支持文件夹展开与折叠，快速定位。
  - 面包屑导航，清晰展示当前路径。
- **拖拽排序**: 基于 `@dnd-kit` 实现，支持通过拖拽自由调整书签和文件夹的顺序。
- **智能排序**:
  - 支持多种排序方式：按名称升序、按名称降序、按名称长度排序。
  - 排序时始终保持文件夹置顶。
- **多重搜索工具**:
  - **全局搜索**: 顶部搜索栏可快速检索所有书签。
  - **自定义搜索引擎**: 右侧面板提供快捷搜索工具，内置 Google, Bing, Baidu，支持自定义添加、编辑和删除搜索引擎。
- **国际化 (i18n)**:
  - 完美支持中文 (Simplified Chinese) 和英文 (English) 界面切换。
  - 自动适配浏览器语言设置。

## 🛠️ 安装与开发指南

### 前置要求

- Node.js (推荐 v18 或更高版本)
- npm

### 开发步骤

1.  **安装依赖**
    在项目根目录下运行：

    ```bash
    npm install
    ```

2.  **开发模式**

    ```bash
    npm run dev
    ```

3.  **构建生产版本**
    构建生成的插件文件将输出到 `dist` 目录：

    ```bash
    npm run build
    ```

4.  **加载到 Chrome 浏览器**
    1. 打开 Chrome 浏览器，在地址栏输入 `chrome://extensions/` 并回车。
    2. 打开右上角的 **"开发者模式" (Developer mode)** 开关。
    3. 点击左上角的 **"加载已解压的扩展程序" (Load unpacked)** 按钮。
    4. 选择本项目根目录下的 `dist` 文件夹（注意：是 build 生成的 `dist` 目录）。

## 📖 使用说明

1.  **打开管理器**: 插件安装成功后，点击浏览器工具栏上的 ZenMarker 图标。
2.  **浏览书签**:
    - 左侧侧边栏展示文件夹树，点击文件夹名称切换主视图内容，点击箭头展开/折叠子文件夹。
    - 主视图以网格形式展示书签，双击文件夹进入，双击书签打开链接。
3.  **排序**: 在主视图的空白区域 **右键点击**，在弹出的菜单中选择 "按名称升序"、"按名称降序" 或 "按名称长度排序"。
4.  **快捷搜索**: 在右侧 "Search Tools" 面板中，选择一个搜索引擎，输入关键词并回车，即可在新标签页中打开搜索结果。
5.  **切换语言**: 点击顶部工具栏的地球图标，可在中文和英文之间切换。

## 📦 技术栈

- **Core**: React 18, TypeScript
- **Build Tool**: Vite
- **Drag & Drop**: @dnd-kit
- **Styling**: Standard CSS (Modular & Variable-based)
- **Icons**: Lucide React

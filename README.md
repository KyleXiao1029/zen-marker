# ZenMarker - Chrome Extension

这是一个遵循苹果现代 UI 风格的谷歌浏览器书签管理扩展。

## 安装指南

1. 打开谷歌浏览器 (Chrome)。
2. 在地址栏输入 `chrome://extensions` 并回车。
3. 在右上角开启 **开发者模式 (Developer mode)**。
4. 点击左上角的 **加载已解压的扩展程序 (Load unpacked)**。
5. 选择本项目的文件夹: `d:\Chrome-Bookmark-Manager`。

## 功能

- **现代化设计**: 毛玻璃效果，圆角设计，流畅交互。
- **书签管理**: 浏览、搜索、新建文件夹、添加当前页。
- **快捷操作**: 点击书签直接在后台新标签页打开，点击文件夹进入。
- **编辑/删除**: 悬浮书签可看到编辑和删除按钮。

##目录结构

- manifest.json: 配置文件的
- popup.html: 界面结构
- style.css: 样式
- popup.js: 逻辑代码
- background.js: 后台服务
